# Medical Service Provider Demo App

The demo app is used by medical service providers to deliver medical care to patients at a distance by means of technology.

## Technologies Implemented

- React Native
- Firebase
- Redux
- Google Maps API
- Audio & Video Calls with [Voximplant](https://voximplant.com/docs/gettingstarted/features)
- Real-Time chat messaging with [Stream Chat](https://getstream.io/chat/sdk/react-native/)
